# AGENTS

This repository does not currently define any custom Copilot Chat agents.

## Purpose

Use this file to document workspace-specific agent definitions, behavior, and usage patterns for contributors. If custom agents are added later, this is the place to explain:

- what each agent does
- when to use each agent
- any workspace conventions or file locations for agent configuration

## Suggested agent files

If you add custom agents, the following files are typically used:

- `.copilot/agents/*.agent.md` or `.copilot/agents/*.prompt.md`
- `.instructions.md` or `.agent.md` files alongside source code
- `AGENTS.md` for a human-readable overview

## Notes

If you want help creating a custom agent for this repository, add details here and the agent behavior can be described in more depth.
